
  # Dark Cybersecurity Portfolio Design
Dark Cybersecurity Portfolio is a modern, responsive personal portfolio built with React and Vite, designed to showcase cybersecurity skills, projects, and professional experience. The site features a dark-themed UI with smooth animations, responsive layouts, and optimized performance, highlighting expertise in networking, penetration testing, web development, cloud computing, and IT support. It serves as a professional online presence for recruiters and collaborators, and is deployed using Vercel for fast, reliable global access.
 

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
